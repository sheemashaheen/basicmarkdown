### Create Movie tickets table

> Movie tickets will have 3 columns

1. Title  
- TEXT
1. Release Year
- YEAR
1. Availability
- TINTINT

Code in SQL

### Creating the data
 

 
 CREATE TABLE `tickets`.`movie tickets` (`TITLE` TEXT NOT NULL , `RELEASE YEAR` YEAR NOT NULL , `AVAILABILITY` TINYINT NOT NULL ) ENGINE = InnoDB;
